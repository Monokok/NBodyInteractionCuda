**Проект решения на С++ для задачи взаимодействия N-тел (параллельный алгоритм)**

Дана система N-тел (частиц, элементов), каждое из которых воздействует на другие элементы. 
Стенки области являются зеркальными: если частица ушла за границы области, то возвращаем её обратно, меняя знак скорости на противоположный.
Состояние частиц и их взаимодействие описывается вторым законом Ньютона.
Массы тел, а также положения и скорости в начальный момент времени считаются известными. 
Необходимо найти положения и скорости всех частиц в произвольный момент времени.

Геометрический вид параллелизма.
Каждый thread отвечает за свое тело (точку, частицу), для которого производит:
- просчет взаимодействия с другими телами;
- обновление координат;
- обработка взаимодействия с границами системы.
